
---

### Add Dependency
```xml
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-devtools</artifactId>
    <optional>true</optional>
</dependency>
```

---

### Enable in IDE Settings

a. **Build project automatically**
   - Settings → Build, Execution, Deployment → Compiler → Check "Build project automatically"

b. **Allow auto-make to start**
   - Settings → Advanced Settings → Check "Allow auto-make to start even if developed application is currently running"

---

### Spring Boot Dev Tools Properties

| Property | Default | Description |
|----------|---------|-------------|
| `spring.devtools.restart.enabled` | `true` | Enables automatic restart when classpath files change |
| `spring.devtools.restart.exclude` | `static/**, public/**` | Excludes specific paths from triggering restart |
| `spring.devtools.restart.additional-paths` | — | Additional directories to monitor for restart |
| `spring.devtools.restart.trigger-file` | — | Restart only when this specific file changes |
| `spring.devtools.restart.poll-interval` | `1000ms` | How often DevTools checks for file changes |
| `spring.devtools.restart.quiet-period` | `400ms` | Time to wait after the last change before restarting |
| `spring.devtools.livereload.enabled` | `true` | Enables LiveReload server for automatic browser refresh |
| `spring.devtools.livereload.port` | `35729` | Port used by LiveReload server |
| `spring.devtools.add-properties` | `true` | Automatically applies development-friendly defaults (e.g., disables template caching) |
