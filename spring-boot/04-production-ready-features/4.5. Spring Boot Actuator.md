### What is Actuator?

Spring Boot Actuator provides out-of-the-box endpoints that expose application health, metrics, beans, and environment info. Useful for monitoring in both development and production.

### Dependency
```xml
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-actuator</artifactId>
</dependency>
```

---

### Key Endpoints

| Endpoint | URL | Description |
|----------|-----|-------------|
| `health` | `/actuator/health` | Shows application health status |
| `info` | `/actuator/info` | Displays app info (configure in properties) |
| `beans` | `/actuator/beans` | Lists all beans in the application |
| `env` | `/actuator/env` | Shows Spring Environment properties |
| `mappings` | `/actuator/mappings` | Lists all `@RequestMapping` paths |
| `threaddump` | `/actuator/threaddump` | Thread dump of the application |
| `metrics` | `/actuator/metrics` | Application metrics (memory, CPU, requests) |
| `loggers` | `/actuator/loggers` | View and change log levels at runtime |

---

### Configuration
```properties
# Expose all endpoints (development only)
management.endpoints.web.exposure.include=*

# Expose specific endpoints (recommended for production)
management.endpoints.web.exposure.include=health,info,metrics

# Change base path (default is /actuator)
management.endpoints.web.base-path=/management
```

**Production warning:** Never expose all endpoints (`*`) in production — it leaks sensitive application internals. Expose only what's needed.
