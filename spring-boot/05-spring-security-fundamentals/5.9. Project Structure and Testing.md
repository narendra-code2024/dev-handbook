### Project Structure

|Package|Class|Purpose|
|---|---|---|
|`config`|`SecurityConfig.java`|SecurityFilterChain, beans|
|`security`|`JwtAuthFilter.java`|JWT filter for every request|
|`service`|`JwtService.java`|Token generation and validation|
|`service`|`CustomUserDetailsService.java`|Bridge between DB and Spring Security|
|`service`|`AuthService.java` / `AuthServiceImpl.java`|Register, login, logout logic|
|`controller`|`AuthController.java`|Auth endpoints|
|`model`|`User.java`|User entity|
|`repository`|`UserRepository.java`|User DB access|
|`mapper`|`UserMapper.java`|Entity ↔ DTO conversion|
|`dto`|`RegisterRequest`, `LoginRequest`, `AuthResponse`, `UserResponse`|Request/Response DTOs|
|`advice`|`GlobalResponseHandler`, `GlobalExceptionHandler`|Response wrapping, error handling|

### Testing Cheatsheet

**Register:**

> `POST /api/auth/register` with body: `{ firstName, lastName, email, password }` → 201 Created + UserResponse

**Login:**

> `POST /api/auth/login` with body: `{ email, password }` → 200 OK + `{ token: "eyJ..." }` + jwt cookie set

**Protected Endpoint (Postman):**

> `GET /api/posts/1` → Authorization tab → Bearer Token → paste token → 200 OK

**Protected Endpoint (SPA):**

> `fetch('/api/posts/1', { credentials: 'include' })` — browser sends jwt cookie automatically

**Logout:**

> `POST /api/auth/logout` → 204 No Content + jwt cookie cleared
